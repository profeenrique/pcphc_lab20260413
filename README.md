## DES01: Tiempo de estudio en casa
Crea un programa que calcule cuánto tiempo estudiaste en un día.

**⬇️ Entrada:**
```
Ingresa las horas de estudio en la tarde: 2
Ingresa las horas de estudio en la noche: 1
Ingresa los minutos extra de repaso: 30
```

**⬆️ Salida esperada:**
```
Estudiaste 3 horas y 30 minutos en total.
Eso equivale a 210 minutos.
```

## DES02: Costo de una salida al cine
Crea un programa que calcule cuánto dinero gastarás en una salida al cine.

**⬇️ Entrada:**
```
Ingresa el nombre de la película: Minecraft
Ingresa el precio de la entrada: 4500
Ingresa la cantidad de entradas: 2
Ingresa el dinero extra para comida: 3000
```

**⬆️ Salida esperada:**
```
Película: Minecraft
Subtotal entradas: $9000
Extra en comida: $3000
Total a pagar: $12000
```

## DES03: ¿Cómo estuvo tu asistencia?
Crea un programa que analice tu asistencia semanal y determine si tuviste buena asistencia. La asistencia es alta si es 80% o más

**⬇️ Entrada:**
```
Nombre del estudiante: Antonia
Días asistidos: 4
Total de días de clases: 5
```

**⬆️ Salida esperada:**
```
Estudiante: Antonia
Porcentaje de asistencia: 80.0
¿Asistencia alta?: True
```

## DES04: Estado de batería del celular
Crea un programa que determine el estado de la batería de un celular según su porcentaje. 

Considera que:

- Batería crítica si tiene menos de 20
- Batería media si tiene entre 20 y 80
- Batería alta si tiene más de 80

**⬇️ Entrada:**
```
Ingresa el porcentaje de batería: 15
```

**⬆️ Salida esperada:**
```
Batería crítica
```
